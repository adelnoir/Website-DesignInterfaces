Projeto desenvolvido no âmbito da unidade curricular de **Design de Interfaces**.

O objetivo é explorar princípios fundamentais de design, composição visual, formatos de imagem, tendências contemporâneas e acessibilidade digital através de uma aplicação web interativa.


## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
  - Grid Layout
  - Flexbox
  - Transições e Animações
  - Backdrop Filter
- **JavaScript (Vanilla)**
  
## Responsividade

O projeto adapta-se automaticamente a:

- Desktop  
- Mobile  
- Tablet  

Media queries garantem reorganização do layout.
